Drop a photo in here named after the category's folder name under
public/products/ (the same slug that appears in the URL, e.g.
/categories/fabric), and it's used as that category card's background
on the homepage automatically.

```
fabric.jpg     → background for the "Fabrics" card
shirts.jpg     → background for the "Apparel" card
```
(`.jpeg`, `.png`, and `.webp` also work — just match the category slug
as the filename.)

Recommended: landscape, roughly 16:10 aspect ratio, at least 1200px
wide — these cards are wide and not very tall.

If a category has no matching file here, that card just uses the
existing woven-pattern background instead, so nothing breaks while
you're still sourcing photos.
